# 🚀 Your Creative AI Toolkit

All-in-one AI app for scripts, stories, translation, thumbnails & voice generation.

---

## 🔥 Features

- 🎬 Recapper (Video → Recap Script)
- ✍️ Content Creator (Social Media Scripts)
- 📖 Story Creator (AI Stories)
- 🎨 Thumbnail Generator
- 🌍 Translate (Multi-language)
- 📝 SRT Subtitle Tool
- 📚 Novel Translator (PDF/EPUB)
- 🔊 AI Voice (Text to Speech)
-("🎨 Thumbnail AI", "thumbnail")
---

## 🚀 How to Use

1. Select a tool from the dashboard  
2. Click "Launch"  
3. Use the feature
